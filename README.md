## Game Overview

!ERRORi is a short 2D platformer set in a collapsing digital world filled with corrupted data, broken geometry, and fading memories.  
Built using the Godot Engine, this game was developed fully solo within the one-week jam period of [Scream Jam 2025](https://itch.io/jam/scream-jam-2025).

---

## Jam Results

- Fully solo-developed (code, art, level design, sound, effects)
- Ranked Top 30% overall: #277 out of 930 entries
